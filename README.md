# anythingToPEM

This is a simple Python script that converts a base64 encoded string to a PEM file. It is useful for converting 
certificates and keys to PEM format.

## Usage

```bash
python anythingToPem.py "<b64 string>"
```

# downloadUrlPems
This is a simple Python script that downloads all the PEM files from a given URL. It is useful for downloading certificates
from websites with non-public CAs.

## Usage

```bash
python downloadUrlPems.py https://example.com https://anotherexample.com
```
